Getting started

Kaigara is an extendable shell command line for managing simple devops tasks. Easily download and install Kaigara using any popular operational system. The setup process will take just couple of minutes.

Please follow the link https://github.com/helios-technologies/kaigara to download

Or install it yourself as: $ gem install kaigara
